# flex-stereo
A binocular camera system with adaptive baseline for biomimetic, rolling-eye stereo vision and depth estimation.

## Environment Setup

### Create conda environment

To set up the required environment for FreeCAD CAD automation:

```bash
conda env create -f environment.yml
```

### Activate the environment

```bash
conda activate freecad-mcp
```

### Verify installation

```bash
python -c "import mcp; import pydantic; print('Environment OK')"
```
